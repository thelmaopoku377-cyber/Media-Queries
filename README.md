# Media Queries Session Setup

Welcome to CSS Media Queries session! Follow these setup steps before class starts:

## Getting Started

1. **Fork this repository** to your own GitHub account using the "Fork" button in the top right.
2. **Clone your forked repository** down to your local machine:
   ```bash
   git clone <your-forked-repo-url>
   ```
3. Open the folder in **VS Code**.
4. Right-click `index.html` and choose **Open with Live Server**.

## Lab Instructions
During class, we will write our responsive logic together using media queries in our `style.css` file. 

At the end of class, you will save, commit, and push your working responsive design to your personal repository!
